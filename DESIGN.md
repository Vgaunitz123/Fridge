# Designsystem – Recept-appen

## Känsla
Premium matapp. Tänk Hallon möter NYT Cooking.
Aldrig generiskt, aldrig AI-aktig.

## Färger
| Token | Värde | Användning |
|---|---|---|
| Bakgrund | `#F5F3EE` | Sidans bakgrund |
| Primär | `#1C3A2A` | Knappar, aktiv nav, accenter |
| Text | `#1A1A1A` | Rubriker och brödtext |
| Muted text | `#6B6B6B` | Labels, metadata |
| Kortbakgrund | `#FFFFFF` | Kort, modaler |

## Typografi
- **Rubriker:** Georgia eller serif, font-weight 400–500
- **Brödtext:** system-ui, 15px, line-height 1.6
- **Etiketter:** 11px, letter-spacing 0.08em, UPPERCASE

## Kort
- Liggande format med riktig foto (16:9 eller 3:2)
- **Aldrig emoji som hero-bild**
- Subtila taggar nere till vänster på bilden
- **Ingen orange** – använd grönt eller neutralt

## Komponenter
- **Sökfält:** tunn border, ingen skugga, pill-form
- **Filterpills:** outline-stil när inaktiv, fylld `#1C3A2A` när aktiv
- **Knappar:** rektangulära, inte för rundade (`border-radius: 8px`)
- **Tab bar:** ren vit, tunna ikoner
